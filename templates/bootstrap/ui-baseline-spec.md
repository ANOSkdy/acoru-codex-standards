# UI baseline spec

## 目的
空repo bootstrap の最小UIは「見栄え」より「build safety」「mobile-first」「説明責任」を優先する。

## `app/page.tsx`
- DB不要
- env不要
- semantic HTML
- short project intro
- setup notes
- links to example API endpoints
- no self-fetch

## CSS
- simple only
- mobile-first
- no fixed desktop-only width
- readable spacing
- visible focus styles
- accessible tap targets
